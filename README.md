# シナチェン！ (Scenario Change Viewer)

## ツールの目的
「シナチェン！」は、新旧のTRPGシナリオ（PDF形式）を比較し、変更された箇所を簡単に確認するためのブラウザ向けツールです。
シナリオの改訂やアップデートが行われた際に、「どこが変わったのか」「重要な判定の数値が変わっていないか」などを素早く把握し、ゲームマスター（KP）などの進行役の負担を減らすことを目的としています。

## 機能概要
* **PDFテキストの差分抽出**: 変更前と変更後の2つのPDFファイルをドラッグ＆ドロップするだけで、テキストの変更点（追加・削除）を行単位で比較します。
* **TRPG特有の重要変更ハイライト**: 「判定」「ダイス（1d3, 1D100など）」「ダメージボーナス（db）」「SAN」といったTRPGで重要なキーワードや、「こぶし」「目星」などの技能名の変更を検知し、「★ TRPGで重要そうな変更」として視覚的に分かりやすく強調表示します。
* **重要箇所のフィルタリング**: チェックボックスをオンにするだけで、膨大な変更点の中から「重要そうな変更」のみを絞り込んで表示することができ、確認作業を大幅に効率化します。
* **ノイズの排除**: PDFのページマーカーのみの変更や、単純な空行の増減など、シナリオの内容に影響を与えない無駄な差分は表示から自動的に除外されます。
* **完全なクライアントサイド処理**: ブラウザ上（ローカル）ですべての解析が完結するため、サーバーにファイルをアップロードする必要がなく、セキュアで高速です。
